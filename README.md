Sistema web que foi desenvolvido com Vue.js e Node.js para gerenciar consultar médicas

FUNCIONALIDADES
- Cadastro de usuários
- Login com autenticação JWT
- Agendamento de consultas
- consulta automatica de endereço através do CEP
- API de clima

TECNLOGIAS UTLIZADAS 
- Vue.js
- Node.js
- express
- MongoDB
- JWT
- API ViaCEP
- API Open-Meteo

  COMO EXECUTAR - BACKEND
  cd backend
  npm install
  npm run dev

  COMO EXECUTAR - FRONTEND
  cd frontend
  npm install
  npm run dev

  ACESSO ATRAVÉS DO ENDEREÇO DE IP : http://localhost:5173
